eraGVT	元気っ子オトコ汎用口上(TS対応)
	by pink@TSF掲示板の刺客
--------------------------------------------------------------------------------

○これは何？

性格：元気っ子のキャラクター用のオトコ汎用口上です。
強制/完全女体化、女体受容による口上変化に対応しています。
（概ね、[女体受容なし→女体受容あり→堕落]の3段階構成）

本口上は、デフォルトで同梱されている元気っ子用口上の改変となっております。

また、おまけとして[\資料\各種テンプレ]フォルダに、TS対応オトコ汎用口上のテンプレ
ファイルを追加します。
（といってもデフォルトのテンプレのキャラ番号を1に変え、本口上から条件分岐をいくつかと
　女体受容時のイベント関数を持ってきただけですが）

○使い方
同梱のERBファイル以下をeraGVTの同ディレクトリにコピーしてください。
（上書きファイルはありません）

その後ゲーム内で、性格：元気っ子のキャラクターにオトコ汎用口上を指定すると、
本口上が使用されます。

○外し方
コピーしたファイルを削除してください。
