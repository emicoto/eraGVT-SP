■eraGVT_別人改造版0.393用　衣装改造パッチv4.03
------------------------------------------------------------------------------------
【パッチ概要】
,;ﾞ ・ω・;,  ＜やっててちょっとかゆいとこに手が届かなかったので弄くり回しました。
フレーバーとは言えやっぱ拘りたいので個人の趣味全開な変更加えちゃった。許して？
あとこれとか欲しくない…？欲しい。って個人的に思った衣装を好き勝手に追加。
あれとこれ被ってね？とかこんな細分化しなくてよくね？って気はしないでもないけど気にしない。

バランスとかちゃんと考えてないのでこうした方がよくね？とかあったら好きにしちゃってください。


【導入方法】
衣装改造フォルダ内のCSVフォルダとERBフォルダを本体に上書きでOK。


・v4.0修正内容
インナーのカスタマイズによる命中補正を削除して触手の油断度補正を追加。
油断度補正の計算の為、14桁目は油断度低下、15桁目は油断度上昇の値として使用。
アウターにスクールスイムストライク追加。
インナーにジュニア下着追加。
インナーにスポーツ下着追加。

あとCLOTH_WEAR.ERBのアウターカスタム関連の関数名がいつの間にかあれこれして変になってたので修正。
前のバージョンでも問題なく動作してたけど差分表示させた時になんか気持ち悪いので…
それと関数名がぱっと見意味不な名前に変わってしまったので…


・v4.01修正内容
水着のカスタマイズ時に油断度表示されないのを修正。
水着のショーツの種類を選択した際の油断度の増減値が説明と異なっていたのを修正。
インナーカスタマイズ時に露出軽減が表示されていなかったので表示するように追加。


・v4.02修正内容
大和撫子の追加装備で天女の羽衣が装備できないのを修正。
大和撫子の追加装備で姫武者を装備すると天女の羽衣まで装備していたのを修正。
大人の下着の油断度の設定値が間違っていたのを修正。


・v4.03修正内容
露出軽減の補正がないインナーでも露出軽減が表示されていたのを修正。
