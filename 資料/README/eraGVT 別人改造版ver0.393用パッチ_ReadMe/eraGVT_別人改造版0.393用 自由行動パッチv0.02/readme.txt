【パッチ概要】
,;ﾞ ・ω・;,  ＜処女ＴＳキャラで援交させるじゃん？ナンパされるじゃん？
　　　　　　　　なんかエロいけど一回しかない上にＴＳ以外がなし崩し的にされるのエロくない？

とか思ってたらなんかいつの間にか妖精さんが作ってくれてたぞ！
tohoKとかmegatenの遠隔調教やら探索やらみたくナンパされたり自由行動中のハプニングエロとか
電車乗ってたら痴漢されたりとかなんかそういう感じのエロにしか行かない行動。

とりあえず未実装部分とかもあるけど一応テスト版リリース。
特に何も考えずに形にする事だけ優先して一つのファイルに全関数に書いちゃってるので
何このクソテキスト？俺にも書かせろよ
みたいなのがやりやすい感じに後々分けたりする予定。


あ、コマンド名変更したので気晴らし外出から自由行動にパッチ名変更しました。


【追加内容】
行動に自由行動が追加されます。
特にメリットはなくただのエロイベントが発生するフレーバー的な行動。
行き先によってナンパされたり痴漢されたりします。

	『街に出る』
　	  街に遊びに行きます。ショッピングしたりゲーセンで遊んだりとか。
　	  ナンパあり。

		【行き先】
		・ショッピングモール
		・アミューズメント施設
		・繁華街
		・公園


	『遠出する』
　	  電車で数駅離れた辺りまで出掛けます。動物園とか水族館とか。
　	  痴漢とナンパ両方が発生する欲張り行動。

		【行き先】
		・動物園
		・水族館
		・遊園地
		・植物園
		・ライブ
		・ウォーターパーク
		・海辺
		・温泉


	『運動する』
　	  運動場やプールで身体を動かします。
　	  何度も同じ場所に足を運んだ場合特定の人物に調教されるイベントが発生。
　	  ナンパあり。でも内容充実してきたりしていらねーなと思ったらこれでナンパ無しにするかも
　	  ＴＳキャラの場合、未変身時と変身時で別々に回数をカウント。

		【行き先】
		・運動公園
		・フィットネスクラブ
		・マッサージサロン
		・プール


	『学校に行く』
　	  学校に行きます。現状行くだけ。
　	  痴漢あり。
　	  素質で小学生/中学生/高校生/大学生を所持していてかつ時間が昼の時のみ実行可能。
　	  ＴＳキャラの場合、変身して初登校するとテキストがちょっと豪華になったり
　	  未変身時と変身時で別々に回数をカウントしたり別々の部活に入れたりちょっとお得。


【導入方法】
本体に上書きでOK。
前回のパッチ導入している場合も同様でOK。
