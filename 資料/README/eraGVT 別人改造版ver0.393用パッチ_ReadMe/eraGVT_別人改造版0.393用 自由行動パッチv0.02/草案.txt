【概要】
ＴＳキャラが処女で援交した時にナンパされるのがなんかエロいけど一回限りのイベントってもったいないなぁ
って感じで日常的になんかエロい目に遭ったりする感じの要素があるとエロくね？
とかなんか思ってたら手を付けてた不思議。

とりあえず作った感じなのでGVT内のイベントコピペったりとかの雑な作り。
原型だけ作って追加分やらテキストやらは丸投げでもいいよね！

eratohoKとかeraMegatenの遠隔調教とかも参考にしてます。

基本的にメリットらしい物がないハイリスクローリターン。
精々魅了経験がちょっと増えるだけ。

ただこれ触手君と戦うってコンセプトから大幅にずれ込む気がするのでそういう意味では無しかもしれない。
どこ行っても急に触手君が襲い掛かって来たのでとかぶち込めば雑に解決出来なくもないか。


【コマンド】
・繁華街
　とりあえず街に出て適当にぶらぶらしてたらナンパされりされなかったり。
　その後持ち帰られたり断ったら逆恨みでレイプされたりとかそういうエロ。
　持ち帰られた後連絡先交換されてセフレにされて調教されたりとかもありかもしれない。

・遠出
　電車に乗ってちょっと遠くに遊びに行こうとしたら痴漢された系な感じで
　車内でエロいことされた後にホテルとか駅のトイレに連れ込まれて二回戦とか
　連絡先抑えられて定期的に呼び出し調教されたりにつなげてもいいかもしれない

・公園
　繁華街と同じ感じでホームレスに持ち帰られたりとか？
　とりあえずいい感じのが思い浮かばなかったので数合わせ的にぶち込んでみた。
　アルバイトで触手君とやってるお姉さんいるし急に触手君が襲いかかって来たとかもありか
　
・学校
　魔法少女なんだし触手君と同じくらい必要だよね？
　折角学生系のタレントあるけど特に活かされてる気がしないから…とかいう理由は後付。
　とりあえず学校に行きます。
　変身を見られて脅されたりとか恋人といちゃついてるのを不純異性交遊扱いで脅されてNTRたりとか
　なんか色々定番のエロに繋げられる気がする。
　変身して性転換する場合は組織の権力で変身後の学籍もあるとかそんな雑さで

基本イベントが発生すると100％回避の目は無し。
ナンパされるとデートを挟んでお持ち帰りされる。
デートの誘いをお断りすると確率でレイプイベントに移行。
お持ち帰り拒否でもデートの誘い以上の確立でレイプ移行。
きつい断り方程ぶち切れレイプになりやすい。

痴漢された場合は抵抗しても確立で失敗。
脅迫も似たような感じでいいかも。

触手くんに従順度上げられてると失敗率が高まるマイナスコンボ。
